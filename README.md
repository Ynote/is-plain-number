# is-plain-number

## Install

Install with npm:
```sh
$ npm install --save is-plain-number
```

## Usage

With CommonJs:
```js
const isPlainNumber = require('is-plain-number');
```

With ECMAScript module:
```js
import isPlainNumber from 'is-plain-number'
```
